# Dashboard de Vendas - Xbox Game Pass
Bootcamp Santander 2025 - Ciência de Dados com Python

Este repositório contém um **Dashboard em Microsoft Excel** desenvolvido a partir de uma base de dados fictícia de assinaturas do **Xbox Game Pass**, criado como parte do Bootcamp Santander 2025 - Ciência de Dados com Python.

O objetivo é aplicar conceitos de **análise de dados**, **visualização** e **storytelling** para responder perguntas de negócio e apresentar insights relevantes.



## Objetivos do Projeto
- Consolidar informações de assinaturas do Xbox Game Pass.
- Criar indicadores de faturamento e vendas de planos.
- Responder perguntas de negócio com base em cálculos e tabelas dinâmicas.
- Construir um dashboard interativo e visualmente atrativo.



## Estrutura do Arquivo
O Excel contém as seguintes seções:

### 1. **Assets**
- Paleta de cores utilizada no dashboard.
- Definições visuais para menus, ícones e logos.

### 2. **Bases**
- Dados de assinantes com informações como:
- ID do assinante
- Nome
- Plano (Core, Standard, Ultimate)
- Data de início
- Auto-renovação
- Preço da assinatura
- Tipo de assinatura (Mensal, Trimestral, Anual)
- Adesão a passes adicionais (EA Play, Minecraft)
- Valor total consolidado

### 3. **Cálculos**
Perguntas de negócio respondidas com tabelas dinâmicas e agregações:
- **Pergunta 1:** Faturamento total de vendas de planos anuais.
- **Pergunta 2:** Faturamento total de planos anuais, separado por auto-renovação.
- **Pergunta 3:** Total de vendas de assinaturas do **EA Play Season Pass**.
- **Pergunta 4:** Total de vendas de assinaturas do **Minecraft Season Pass**.

### 4. **Dashboard**
- Painel visual consolidado:
- Gráficos de faturamento por tipo de plano.
- Comparativo de assinaturas com e sem auto-renovação.
- Indicadores de vendas de passes adicionais.
- Paleta de cores personalizada para consistência visual.



## Principais Insights
- O plano **Ultimate** concentra a maior parte das receitas, especialmente em assinaturas anuais.
- A adesão ao **EA Play Season Pass** e ao **Minecraft Season Pass** é significativa entre assinantes Ultimate e Standard.
- Assinaturas com **auto-renovação** representam uma parcela relevante do faturamento recorrente.



## Como Utilizar
1. Baixe o arquivo Excel disponível neste repositório.
2. Explore as abas **Bases**, **Cálculos** e **Dashboard**.
3. Interaja com os gráficos e tabelas dinâmicas para visualizar os insights.



## Tecnologias Utilizadas
- **Microsoft Excel** (Tabelas Dinâmicas, Gráficos, Formatação Condicional).
- **Python (Bootcamp Santander 2025)** para análise complementar e manipulação de dados.


## Conclusão
Este projeto demonstra como **dados brutos podem ser transformados em insights acionáveis** através de ferramentas simples como o Excel, servindo como base para análises mais avançadas em Python e outras linguagens de ciência de dados.
